Welcome Summoner!
The first demo version of RuneBook is out! RuneBook can synchronize with League Client. You can see the connection status in the bottom right of this window. Hover over the badge to see the status description. The interface for now shows a button and a dropdown menu. The latter is synchronized with your rune pages. It will always show the last selected rune page and it updates automatically whenever you change something on the client. Also, you can select a different page from the dropdown, and the client will instantly reflect the change. Instead, the SET button is made to try page insertions and deletions: when pressed, the current selected rune page will be deleted, if applicable, and it will be replaced with a test page. Again, all the changes are synchronized between RuneBook and LoL Client.
